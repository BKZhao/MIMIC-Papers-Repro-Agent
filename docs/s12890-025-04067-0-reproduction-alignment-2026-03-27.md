## 4. 为什么还没完全对齐

当前最主要的 gap 在队列构建层，而不是统计代码是否运行。

- 队列规模不一致（主因）:
  - 论文报告 N=559
  - 当前复现 N=739
  - 差值 +180
- 论文中的一些关键细节仍不完整:
  - ARF ICD 精确代码集合在文中并未完全展开
  - 多重插补细节与建模联动（插补模型、重复次数、池化策略）尚未复刻
  - LASSO 的“变量候选池 -> 最终 9 变量”路径尚未完全复刻
  - 500 次 bootstrap 校准尚未实现为同口径报告
- 因此目前属于:
  - 方法族方向对齐（Cox/ROC/Calibration/DCA + train/validation）
  - 论文关键流程细节已部分补齐（缺失阈值与插补）
  - 队列口径与验证性能仍待收敛
